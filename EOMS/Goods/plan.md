# 商品管理项目计划管理

## 概述

详尽的项目规划可查看：[项目排期表](http://192.168.10.138/plan/goods/)

## 分期及目标

相关分期目标可见：[分期拆分表](https://my.mindnode.com/71Hr5xHzdZwubX8Wz1ocXP7qV58wUyjW4sZqc7sp)

### 第1期

第一期：数据结构对接及调整

1. 完成ERP数据导入和修改
2. YPOSX 更新商品数据对接
3. 店长工具更新商品数据对接
4. YMC及其商品相关业务调整数据
5. 灰度测试门店，验证数据可靠性

### 第2期

第二期：商品套餐管理完成

1. 套餐的数据结构支持
2. 套餐后台管理项
3. 套餐分门店管理（还未赋能门店）
4. 套餐相关业务和财务数据

### 第3期

第三期：商品管理链路及后台

1. 商品管理后台图形界面
2. 新建商品的创建链路

### 第4期

第四期：赋能门店自行管理商品及套餐

1. 门店店长工具菜单创建和管理
2. 门店店长工具内商品的基本管理
3. 门店店长工具内自行管理套餐
