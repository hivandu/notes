# UI&UX 规范

## 概述

EOMS系统后台系统框架如下:

前端技术框架: React + TypeScript + Redux
前端UI组件: [Ant Design](https://ant.design/docs/spec/introduce-cn)

而**设计稿当遵循Ant Design的设计规范**, 在此基础上构建网鱼自己的组件库.
